# ADR-003

Status: Accepted
Data: 2026-05-21

## Contexto
O projeto usa persistencia simples para auth/admin e nao define modelos de dominio. [Fonte: codigo]

## Decisao
Usar SQLite como banco de dados padrao (db.sqlite3). [Fonte: codigo]

## Alternativas consideradas
- Postgres/MySQL. [Inferencia]

## Consequencias positivas
- Setup simples para desenvolvimento e deploy leve. [Inferencia]

## Consequencias negativas
- Limitacoes para concorrencia e escala. [Inferencia]

## Impacto tecnico
- DATABASES configurado para sqlite3 em settings. [Fonte: codigo]

## Quando revisar
Quando houver necessidade de escala, multiusuario ou alta disponibilidade. [Inferencia]
